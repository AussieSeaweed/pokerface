PokerFace Package
=================

Classes, functions, and variables in PokerFace are documented below. All
of these can be imported from the top-level PokerFace module.

pokerface.game module
---------------------

These classes contains core logic of poker games. They are highly
customizable, allowing you to combine various utility classes such as
decks, stages, evaluators, et cetera to define and create custom poker
games.

The :class:`pokerface.game.PokerGame` class, the
:class:`pokerface.game.PokerNature` class, and the
:class:`pokerface.game.PokerPlayer` classes inherit from sequential game
classes defined in the `Krieg <https://krieg.readthedocs.io/>`_ package.
